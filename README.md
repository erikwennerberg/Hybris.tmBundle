# Hybris.tmBundle
Textmate Bundle for Hybris. Should work with any version of TextMate on Mac OS.

V0.2/09212015:
- language grammar for Impex.

1) Download as a zip

2) Unzip

3) Double click on the .tmBundle file 

4) Follow instructions to install the bundle

5) Enjoy
